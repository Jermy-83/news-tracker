# newstracker

Private market news tracker desktop app.

## Run the desktop app

```powershell
npm start
```

This launches the local server and opens the app in a desktop window.

## Run in web mode

```powershell
npm run start:web
```

Then open:

`http://127.0.0.1:3180/`

## What it does

- Tracks market-moving news by watchlist
- Separates fresh items from visible items
- Sorts by latest, oldest, score, confidence, or source count
- Shows a summary plus an overall takeaway for the selected article
