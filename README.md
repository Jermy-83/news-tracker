# newstracker

Private market news tracker app.

## Run locally

```powershell
node server.js
```

Then open:

`http://127.0.0.1:3180/`

## What it does

- Tracks market-moving news by watchlist
- Separates fresh items from visible items
- Sorts by latest, oldest, score, confidence, or source count
- Shows a summary plus an overall takeaway for the selected article
